### Bài 1: Cài đặt 3 thuật toán ⭐⭐
Bubble Sort, Selection Sort, Insertion Sort. Mỗi cái in từng bước thay đổi mảng.
câu trả lời:
#include <iostream>
#include <vector>
using namespace std;

void printArray(const vector<int>& a) {
    for (int x : a) cout << x << " ";
    cout << endl;
}

// Bubble Sort
void bubbleSort(vector<int> a) {
    cout << "\n=== Bubble Sort ===\n";

    int step = 1;

    for (int i = 0; i < a.size() - 1; i++) {
        bool swapped = false;

        for (int j = 0; j < a.size() - i - 1; j++) {
            if (a[j] > a[j + 1]) {
                swap(a[j], a[j + 1]);
                swapped = true;

                cout << "Buoc " << step++ << ": ";
                printArray(a);
            }
        }

        if (!swapped) break;
    }
}

// Selection Sort
void selectionSort(vector<int> a) {
    cout << "\n=== Selection Sort ===\n";

    int step = 1;

    for (int i = 0; i < a.size() - 1; i++) {
        int minIndex = i;

        for (int j = i + 1; j < a.size(); j++) {
            if (a[j] < a[minIndex])
                minIndex = j;
        }

        if (minIndex != i) {
            swap(a[i], a[minIndex]);

            cout << "Buoc " << step++ << ": ";
            printArray(a);
        }
    }
}

// Insertion Sort
void insertionSort(vector<int> a) {
    cout << "\n=== Insertion Sort ===\n";

    int step = 1;

    for (int i = 1; i < a.size(); i++) {
        int key = a[i];
        int j = i - 1;

        while (j >= 0 && a[j] > key) {
            a[j + 1] = a[j];
            j--;
        }

        a[j + 1] = key;

        cout << "Buoc " << step++ << ": ";
        printArray(a);
    }
}

int main() {
    vector<int> a = {5, 2, 8, 1, 3};

    bubbleSort(a);
    selectionSort(a);
    insertionSort(a);

    return 0;
}
### Bài 2: Đếm số phép so sánh và hoán vị ⭐⭐
Với cùng 1 mảng 100 phần tử: đếm số lần so sánh và số lần hoán vị của 3 thuật toán. In bảng so sánh.

#include <iostream>
#include <vector>
#include <iomanip>
using namespace std;

struct Result {
    long long comparisons = 0;
    long long swaps = 0;
};

Result bubbleSort(vector<int> a) {
    Result r;

    for (int i = 0; i < a.size() - 1; i++) {
        for (int j = 0; j < a.size() - i - 1; j++) {
            r.comparisons++;

            if (a[j] > a[j + 1]) {
                swap(a[j], a[j + 1]);
                r.swaps++;
            }
        }
    }

    return r;
}

Result selectionSort(vector<int> a) {
    Result r;

    for (int i = 0; i < a.size() - 1; i++) {
        int minIndex = i;

        for (int j = i + 1; j < a.size(); j++) {
            r.comparisons++;

            if (a[j] < a[minIndex])
                minIndex = j;
        }

        if (minIndex != i) {
            swap(a[i], a[minIndex]);
            r.swaps++;
        }
    }

    return r;
}

Result insertionSort(vector<int> a) {
    Result r;

    for (int i = 1; i < a.size(); i++) {
        int key = a[i];
        int j = i - 1;

        while (j >= 0) {
            r.comparisons++;

            if (a[j] > key) {
                a[j + 1] = a[j];
                r.swaps++;
                j--;
            } else {
                break;
            }
        }

        a[j + 1] = key;
    }

    return r;
}

int main() {
    vector<int> arr;

    for (int i = 100; i >= 1; i--)
        arr.push_back(i);

    Result s = selectionSort(arr);
    Result i = insertionSort(arr);

### Bài 3: Best / Worst / Average Case ⭐⭐
Với mảng đã sắp xếp, ngược chiều, và ngẫu nhiên — đo thời gian chạy 3 thuật toán. Kết luận khi nào nên dùng cái nào.
câu trả lời:
#include <iostream>
#include <vector>
#include <algorithm>
#include <random>
#include <chrono>

using namespace std;
using namespace chrono;

void bubbleSort(vector<int>& a) {
    for (int i = 0; i < a.size() - 1; i++)
        for (int j = 0; j < a.size() - i - 1; j++)
            if (a[j] > a[j + 1])
                swap(a[j], a[j + 1]);
}

void selectionSort(vector<int>& a) {
    for (int i = 0; i < a.size() - 1; i++) {
        int minIndex = i;

        for (int j = i + 1; j < a.size(); j++)
            if (a[j] < a[minIndex])
                minIndex = j;

        swap(a[i], a[minIndex]);
    }
}

void insertionSort(vector<int>& a) {
    for (int i = 1; i < a.size(); i++) {
        int key = a[i];
        int j = i - 1;

        while (j >= 0 && a[j] > key) {
            a[j + 1] = a[j];
            j--;
        }

        a[j + 1] = key;
    }
}

template<typename Func>
double measure(Func sortFunc, vector<int> arr) {
    auto start = high_resolution_clock::now();

    sortFunc(arr);

    auto end = high_resolution_clock::now();

    return duration<double, milli>(end - start).count();
}

int main() {
    const int N = 5000;

    vector<int> best(N);
    for (int i = 0; i < N; i++) best[i] = i;

    vector<int> worst = best;
    reverse(worst.begin(), worst.end());

    vector<int> average = best;
    shuffle(average.begin(), average.end(),
            mt19937(random_device{}()));

    cout << "=== BEST CASE ===\n";
    cout << "Bubble: " << measure(bubbleSort, best) << " ms\n";
    cout << "Selection: " << measure(selectionSort, best) << " ms\n";
    cout << "Insertion: " << measure(insertionSort, best) << " ms\n";

    cout << "\n=== WORST CASE ===\n";
    cout << "Bubble: " << measure(bubbleSort, worst) << " ms\n";
    cout << "Selection: " << measure(selectionSort, worst) << " ms\n";
    cout << "Insertion: " << measure(insertionSort, worst) << " ms\n";

    cout << "\n=== RANDOM CASE ===\n";
    cout << "Bubble: " << measure(bubbleSort, average) << " ms\n";
    cout << "Selection: " << measure(selectionSort, average) << " ms\n";
    cout << "Insertion: " << measure(insertionSort, average) << " ms\n";
}
### Bài 4: 🔥 Dự Án Mini — Sorting Visualizer (Console) ⭐⭐⭐
> **Cảm hứng:** [Sorting — algorithm-visualizer.org](https://algorithm-visualizer.org/simple-recursive/bubble-sort)
câu trả lời:
#include <iostream>
#include <vector>
#include <thread>
#include <chrono>

using namespace std;

const string RED = "\033[31m";
const string GREEN = "\033[32m";
const string YELLOW = "\033[33m";
const string CYAN = "\033[36m";
const string RESET = "\033[0m";

void clearScreen()
{
#ifdef _WIN32
    system("cls");
#else
    system("clear");
#endif
}

void drawBars(
    const vector<int>& arr,
    int highlight1,
    int highlight2,
    string algorithm,
    int currentStep,
    int totalSteps,
    long long comparisons,
    long long swaps)
{
    clearScreen();

    cout << CYAN
         << algorithm
         << " - Buoc "
         << currentStep
         << "/"
         << totalSteps
         << RESET
         << "\n\n";

    for (int i = 0; i < arr.size(); i++)
    {
        if (i == highlight1 || i == highlight2)
            cout << RED;

        for (int j = 0; j < arr[i]; j++)
            cout << "█";

        cout << " " << arr[i];

        if (i == highlight1 || i == highlight2)
            cout << RESET;

        cout << "\n";
    }

    cout << "\nDang so sanh: ["
         << highlight1
         << "] va ["
         << highlight2
         << "]\n";

    cout << "So buoc con lai: "
         << totalSteps - currentStep
         << "\n";

    cout << "Comparisons: "
         << comparisons
         << "\n";

    cout << "Swaps/Shifts: "
         << swaps
         << "\n";
}

//////////////////////////////////////////////////
// Bubble Sort
//////////////////////////////////////////////////

void bubbleVisualizer(vector<int> arr)
{
    int n = arr.size();

    int totalSteps = n * (n - 1) / 2;

    long long comparisons = 0;
    long long swaps = 0;
    int step = 1;

    for (int i = 0; i < n - 1; i++)
    {
        for (int j = 0; j < n - i - 1; j++)
        {
            comparisons++;

            drawBars(
                arr,
                j,
                j + 1,
                "Bubble Sort",
                step++,
                totalSteps,
                comparisons,
                swaps);

            this_thread::sleep_for(
                chrono::milliseconds(200));

            if (arr[j] > arr[j + 1])
            {
                swap(arr[j], arr[j + 1]);
                swaps++;
            }
        }
    }

    cout << GREEN
         << "\nDa sap xep xong!\n"
         << RESET;
}

void selectionVisualizer(vector<int> arr)

                minIndex = j;
        }

        if (minIndex != i)
        {
            swap(arr[i], arr[minIndex]);
            swaps++;
        }
    }

    cout << GREEN
         << "\nDa sap xep xong!\n"
         << RESET;
}

//////////////////////////////////////////////////
// Insertion Sort
//////////////////////////////////////////////////

void insertionVisualizer(vector<int> arr)
{
    int n = arr.size();

    int totalSteps = n * (n - 1) / 2;

    long long comparisons = 0;
    long long shifts = 0;
    int step = 1;

    for (int i = 1; i < n; i++)
    {
        int key = arr[i];
        int j = i - 1;

        while (j >= 0)
        {
            comparisons++;

            drawBars(
                arr,
                j,
                j + 1,
                "Insertion Sort",
                step++,
                totalSteps,
                comparisons,
                shifts);

            this_thread::sleep_for(
                chrono::milliseconds(200));

            if (arr[j] > key)
            {
                arr[j + 1] = arr[j];
                shifts++;
                j--;
            }
            else
            {
                break;
            }
        }

        arr[j + 1] = key;
    }

    cout << GREEN
         << "\nDa sap xep xong!\n"
         << RESET;
}
